## Sobre
O Pathfinder RPG é uma ferramenta de simulação de busca de caminhos em ambientes de jogos de mesa. O projeto utiliza uma <b>matriz MxN</b>  para representar um mapa com diferentes biomas (Grama, Floresta, Pântano, etc.), onde cada terreno possui um custo de movimentação específico.
<br><br>
O objetivo principal é demonstrar a aplicação do <b>Algoritmo de Dijkstra</b> e da <b>BFS (Busca em Largura)</b> para encontrar o menor caminho de um nó origem até um nó destino, e verificar se para ambos os casos a resposta é a mesma.
